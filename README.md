# NASA Space Explorer

An immersive space exploration web application built with **React**, **Vite**, and **Tailwind CSS**, powered by **NASA Open APIs**. Explore NASA's Astronomy Picture of the Day, discover fascinating Near-Earth Objects, and experience the beauty of space through a modern, responsive interface.

## Live Demo

🔗 https://nasa-space-explorer-task1.vercel.app/

---

#  Features

### Home

- Modern space-inspired landing page
- Interactive hero section
- Solar System overview
- Featured exploration sections
- Responsive navigation

### Near-Earth Objects

- Browse Near-Earth Objects (NEOs)
- Date-based asteroid search
- Hazardous asteroid identification
- Asteroid details using NASA Lookup API
- Physical and orbital information

### APOD Gallery

- NASA Astronomy Picture of the Day integration
- Browse high-resolution astronomy images
- Date-based APOD search
- Save favourite astronomy images

### Favorites

- Save favourite APOD images
- Persistent favourites using Local Storage
- Quick access to saved images

### UI & Experience

- Fully responsive design
- Modern space-inspired interface
- Reusable component architecture

# Getting Started

### Clone the repository

```bash
git clone (https://github.com/being-shivalika/NASA-space-explorer-Task1.git)
```

### Navigate into the project

```bash
cd nasa-space-explorer
```

### Install dependencies

```bash
npm install
```

### Create a `.env` file

```env
VITE_NASA_API_KEY=YOUR_NASA_API_KEY
```

### Start the development server

```bash
npm run dev
```
